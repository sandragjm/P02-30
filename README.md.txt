1. Hemos clonado las carpetas carpetaCasa y carpetaInstituto para trabajar con ellas.
2. En el caso del segundo apartado, al añadir una clase nueva al repositorio online sin hacer pull, nos dará erros, por lo que para solucionarlo deberemos hacer un git pull antes de realizar nuestra operación.
3. Al no hacer git pull, nuestro repositorio local está más desactualizado que nuestro repositorio online por lo que nos dará erros. Para solucionarlo, debemos usar git pull para tener ambos repositorios (carpetaCasa/Instituto) actualizados.
4. En el caso cuatro pasa básicamente lo mismo, nuestros repositorios no han sido actualizados por lo que al hacer nuestras modificaciones e intentar subirlo, nos dará error.
